TensorFlow 是一个开放源代码机器学习框架。
它可以安装在64位 wintows Ubuntu macOS 上，和 python3.x 工作。它也可以工作在 c 或 java 、 go ，本文只涉及 python 上的使用。

安装 TensorFlow

启动终端
要安装仅支持 CPU 的 TensorFlow 版本，输入以下命令：
pip3 install –upgrade tensorflow
要安装 GPU 版本的 TensorFlow，输入以下命令：
pip3 install –upgrade tensorflow-gpu

验证安装的正确性

import tensorflow as tf
hello = tf.constant(‘Hello, TensorFlow!’)
sess = tf.Session()
print(sess.run(hello))
