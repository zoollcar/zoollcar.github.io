Python是一种高级编程语言，比较贴近于用户使用而不是硬件；它使用简洁的语法，以缩进表示语法结构；
Python是简单，易学的，可以使用多种编程模式（面型对象，函数式编程）

本教程将用最少的文字，尽量覆盖 Python3 的基础语法，适合于学过其他编程语言的同学快速掌握Python和以前学过Python的人快速复习

综述

Python的官方网站https://www.python.org/ 官方文档https://docs.python.org/3/

Python可以在官方下载链接 https://www.python.org/downloads/ 中下载二进制安装包，或通过源码安装，安装完成之后可以通过在命令行中使用python -V显示Python版本

在命令行中直接执行python可以打开Python命令行，并执行任何python命令，其中“ _ ”表示上一命令的结果。
Python源程序的扩展名默认为.py，若要执行.py文件，在命令行中输入python filename.py。
使用tab或空格进行语法缩进
