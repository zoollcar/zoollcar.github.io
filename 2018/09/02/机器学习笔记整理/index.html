机器学习笔记整理

机器学习的理论步骤：

  1. 收集数据
  2. 确定模型
  3. 根据数据训练模型

机器学习的实际步骤

  1. 数据收集
  2. 数据预处理
  3. 设计假设函数
  4. 设计损失函数
  5. 设计训练算法
  6. 进行训练，观察训练过程
  7. 用模型做出预测

    综述

数据

数据是一系列从定义向量组到值域向量组的映射

模型（假设函数）

模型是用来拟合上面那条映射的模型

模型的主要特点是

  1. 拟合：模型可以拟合已有的数据之间的映射关系
  2. 预测（泛化）：对未知数据具有预测的能力

    学习

机器学习的重点就是从数据中得到模型的过程，这个过程叫做学习（训练）

假设函数

从数学上经常用假设函数

计算机尤其是人工神经网络上经常用模型

假设函数使用数学方法描述自变量与因变量之间的关系

假设函数可分为线性和非线性，而实现方式我只学过人工神经网络（惭愧）

人工神经网络

人工神经网络早期受到生物学的影响，但现在已发展成为完全的数学模型，不过模仿生物有时也能产生不错的效果

一个神经元（感知器）的数学模型

对生物上的神经元进行简化可以得到神经元的数学模型

一个神经元只有一个输出，可以有多个输入。输出取决于输入、输入权重、阈值和神经元变换函数

数学模型如下：

$$O_{j}(t+1) = f(\sum_{i=1}^{n}w_{ij}x_{i}(t)-T_{j})$$

其中

  1. j是一个神经元

  2. $$O_{j}(t)$$ 是t时刻j的输出

  3. $$w_{ij}$$ 是i到j的权重（i是另一个上层的另一个神经元)

  4. $$T_{j}$$是j的阈值

  5. $$f()$$是神经元变换函数
