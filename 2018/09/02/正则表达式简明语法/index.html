字符组[ ] 与 任意字符 “.”

[ABC]表示A或B或C
字符组支持由连字符“ - ”来表示一个范围，如[a-z] [1-5]
[^ABC] 排除型字符组。排除后面的字符，也可以是[^a-z]表示除小写字母之外
“.”可以匹配任意一个字符

重复字符 “ * ” 与 重复区间{ }

“*”重复前面的一个字符，0次及以上
{ }可限制一个数字范围，重复前面一个字符

“{“和”}”在shell中有特殊意义，这里需要用到转义字符”\”
